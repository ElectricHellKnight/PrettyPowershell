# PrettyPowershell #

Not much to say, this is the powershell prompt I use. Adds a little color to the prompt, execution time of prior command,
and a warning if you are running as admin. Also changes the taskbar status to your current working directory, which is nice
if you have a lot of Windows open.

Mostly ripped off from: https://commandline.ninja/customize-pscmdprompt/ but I made a few tweaks.
